# 📸 CodeAlpha Image Gallery

## 📖 Project Description

The Image Gallery is a responsive web application developed using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

This project provides an interactive and visually appealing gallery where users can browse images, view them in a full-screen lightbox, and navigate between images using buttons or keyboard controls. The gallery is designed with a modern Glassmorphism user interface and smooth animations to enhance user experience.

---

## 🎯 Objectives

* Create a responsive image gallery using HTML and CSS.
* Implement image navigation using JavaScript.
* Add an attractive and user-friendly interface.
* Provide a lightbox view for enlarged image previews.
* Ensure compatibility across desktop, tablet, and mobile devices.

---

## ⚙️ How It Works

### Step 1

The gallery displays multiple image cards in a responsive grid layout.

### Step 2

When a user clicks an image, JavaScript opens the image inside a lightbox.

### Step 3

Users can navigate between images using:

* Previous Button
* Next Button
* Keyboard Arrow Keys

### Step 4

The lightbox can be closed by:

* Clicking the Close Button
* Pressing the Escape Key
